This module requires the use of JSON Manager v9.8.3

This version only provides some operations that can be modified, but it greatly simplifies the operation
How to use:
The usage method is to click on the desired operation, then click on the add operation, or click on the existing operation to delete it, save and generate a copy of the configuration file, and finally drag and drop it to the JSON Manager.

Attention:
Evasion has been changed to: press once to dodge, long press to roll.
I don't recommend you modify LB+A and LB+B, because when you press these key combinations, the operations of A and B will be triggered first, followed by LB+A and LB+B.
